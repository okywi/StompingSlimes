# Stomping Slimes
A tile based 2D platformer written in Java using the libgdx game development library.

![start menu](platformer.png)

## Features
- [ ] Multiple levels
- [ ] Level picker
- [ ] Pause menu
- [ ] Enemies
- [ ] Tiling system

## How to use
```
WASD to walk  
Spacebar to jump  
ESC to pause
R to restart

Stomp on the slimes! c:
```
## Run
**Important: You need to use Java 17**
```
./gradlew desktop:run

```
